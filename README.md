# Pomodoro

## Installing

1. Clone the Repo
2. Browse to the Pomodoro folder
3. Install applicatication by running: 
    ```sh
    $ npm install
    ```
4. To start the pomodoro-CLI run:
    ```sh
    $ npm start
    ```
5. With you want a CLI command to be installed globaly:
    ```sh
    $ npm install -g
    ```
